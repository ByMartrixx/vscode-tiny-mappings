# Tiny Mappings for VSCode

## Features
- Syntax highlighting on Tiny V2 Mapping files (`.tiny` extension)
- Syntax highlighting on Enigma Mapping files (`.mapping` extension)

## Planed features
- Error checking on `.tiny` and `.mapping` files
